# RAG Knowledge Assistant

AI-powered PDF Q&A app built with LangChain, OpenAI, and Streamlit.

## Features
- Upload any PDF
- Ask questions about it
- Get AI-powered answers

## Tech Stack
- Python
- LangChain
- OpenAI GPT
- Streamlit
